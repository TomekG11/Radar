# UkrainskiReader

Minecraft 1.20.1 Fabric mod - zaawansowany gear reader dla graczy PvP.

## Funkcje

- 🔍 Wykrywanie graczy z netherytowym ekwipunkiem w zasięgu
- 🪶 Opcjonalne wykrywanie elytry
- ⭐ Watchlist - śledź wybranych graczy
- ⛔ Blacklist - ignoruj niechcianych graczy
- 📊 Szczegółowe statystyki HP/armor/absorption
- 🎨 Konfigurowalne motywy kolorystyczne
- ⚡ Tryb wydajności (performance mode)
- 🔔 Powiadomienia o znikniętych graczach (RTP)

## Instalacja

1. Zainstaluj [Fabric Loader](https://fabricmc.net/use/)
2. Zainstaluj [Fabric API](https://modrinth.com/mod/fabric-api)
3. Pobierz najnowszą wersję z [Releases](https://github.com/yourusername/ukrainskireader/releases)
4. Umieść plik `.jar` w folderze `mods`

## Użycie

- **Klawisz G** - otwórz GUI
- **ESC** - zamknij GUI
- Kliknij ⚙️ w prawym górnym rogu GUI aby otworzyć ustawienia
